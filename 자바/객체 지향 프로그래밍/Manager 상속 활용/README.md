# 설명
```
static Manager lectureMgr = new Manager();
```
Manager 클래스는 여러 객체(강의, 학생 등)를 관리하는 역할을 한다. 여기서 lectureMgr는 강의(Lecture) 데이터를 관리할 매니저 객체이다.

```
lectureMgr.readAll("lecture.txt", new LectureFactory());
lectureMgr.printAll();
```
파일의 각 줄을 읽어 Lecture 객체를 만든 뒤 mList에 저장하고 출력한다. 여기 LectureFactory는 Lecture 객체를 생성하는 Factory 클래스다.

```
class LectureFactory implements Factory {
    public Manageable create(Scanner scan) {
        return new Lecture();
    }
}
```

```
public void readAll(String filename, Factory fac) {
    Scanner filein = openFile(filename);
    while (filein.hasNext()) {
        Manageable m = fac.create(filein);  // ★ 여기서 create() 호출
        m.read(filein);
        mList.add(m);
    }
    filein.close();
}
```
Manager의 readAll 함수에서 create 함수를 호출하여 Lecture 객체를 반환한다.
